# GuitarBuddy
A virtual guitar leveraging the Karplus-Strong Algorithm

"q2we4r5ty7u8i9op-[=zxdcfvgbnjmk,.;/' "
Each key represents a note, so:
q = a;
2 = a sharp / b flat;
w = b;
e = c;
4 = c sharp / d flat;
...
' = a sharp / b flat;
(space) = b

Working mechanisms in GuitarString.java and RingBuffer.java files
